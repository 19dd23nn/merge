## Desired Outcome

- Start team mode with concrete runtime evidence, or identify the exact precondition preventing it.
- Determine whether any team-based path can overcome the current git push blockers.
- Push the branch if environment constraints allow it.

- `omx` is installed at `/opt/homebrew/bin/omx`.
- `omx doctor --team` reports no generic team runtime issues.
- `tmux -V` returns `tmux 3.6a`.
- `$TMUX` is empty in the current leader environment.

- `.git` metadata writes are blocked in this sandbox.
- Team skill requires OMX runtime orchestration, not plain in-process fanout.
